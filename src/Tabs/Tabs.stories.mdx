# Tabs

### Simple responsive horizontal navigation **tabs**, with different styles

[Bulma Documentation](https://bulma.io/documentation/components/tabs/)

Supported Versions: `0.9.3`, `0.9.2`, `0.9.1`, `0.9.0`

## Default

The Bulma tabs are a straightforward navigation component that come in a variety of versions. They only require the following structure:

- `<Tabs>` container
  - `<Tabs.List>`: the unordered list
    - `<Tabs.ListItem>`: each list item
      - `<Tabs.Link>`: renders as an anchor element for each link

The default tabs style has a single border at the bottom.

## Styles

If you want a more classic style with **borders**, just append the `boxed` prop.

<Canvas>
  <Story name="Boxed">
    <Tabs boxed>
      <Tabs.List>
        <Tabs.ListItem active>
          <Tabs.Link>Pictures</Tabs.Link>
        </Tabs.ListItem>
        <Tabs.ListItem>
          <Tabs.Link>Videos</Tabs.Link>
        </Tabs.ListItem>
        <Tabs.ListItem>
          <Tabs.Link>Documents</Tabs.Link>
        </Tabs.ListItem>
      </Tabs.List>
    </Tabs>
  </Story>
</Canvas>

If you want **mutually exclusive** tabs (like radio buttons where clicking one deselects all other ones), use the `toggle` prop.

<Canvas>
  <Story name="Toggle">
    <Tabs toggle>
      <Tabs.List>
        <Tabs.ListItem active>
          <Tabs.Link>Pictures</Tabs.Link>
        </Tabs.ListItem>
        <Tabs.ListItem>
          <Tabs.Link>Videos</Tabs.Link>
        </Tabs.ListItem>
        <Tabs.ListItem>
          <Tabs.Link>Documents</Tabs.Link>
        </Tabs.ListItem>
      </Tabs.List>
    </Tabs>
  </Story>
</Canvas>
