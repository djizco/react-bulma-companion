# Title and Subtitle

### Simple **headings** to add depth to your page

[Bulma Documentation](https://bulma.io/documentation/elements/title/)

Supported Versions: `0.9.4`, `0.9.3`, `0.9.2`, `0.9.1`, `0.9.0`

## Default

There are **2 types** of `<Title>` components. Use the `subtitle` prop to get a subtitle.

<Canvas>
  <Story name="Default">
    <React.Fragment>
      <Title>Title</Title>
      <Title subtitle>Subtitle</Title>
    </React.Fragment>
  </Story>
</Canvas>

## Spacing

When you **combine** a title and a subtitle, they move closer together.

As a rule of thumb, it is recommended to use a size difference of **two**. So if you use a `title 1`, combine it with a `subtitle 3`.

You can maintain the normal spacing between titles and subtitles by using the `spaced` prop on the first element.

<Canvas>
  <Story name="Spacing">
    <React.Fragment>
      <Block>
        <Title size="1">Title 1</Title>
        <Title size="3" subtitle>Subtitle 3</Title>
      </Block>
      <Block>
        <Title size="1" spaced>Title 1</Title>
        <Title size="3" subtitle>Subtitle 3</Title>
      </Block>
    </React.Fragment>
  </Story>
</Canvas>

## Header Tags

By default, the `<Title>` component will render as a `p` tag.  If you want to use the native header tags, such as `h1`, use the `component` prop.
