# Card

### An all-around flexible and composable component

[Bulma Documentation](https://bulma.io/documentation/components/card/)

Supported Versions: `0.9.1`, `0.9.0`

## Default

The `<Card>` comprises several components that you can mix and match:

- `<Card>`: the main container
  - `<Card.Header>`: a horizontal bar with a shadow
    - `<Card.HeaderTitle>`: a left-aligned bold text
    - `<Card.HeaderIcon>`: a placeholder for an icon
  - `<Card.Image>`: a fullwidth container for a responsive image
  - `<Card.Content>`: a multi-purpose container for any other element
  - `<Card.Footer>`: a horizontal list of controls
    - `<Card.FooterItem>`: a repeatable list item
