# Textarea

### The multiline **textarea** and its variations

[Bulma Documentation](https://bulma.io/documentation/form/textarea/)

Supported Versions: `0.9.4`, `0.9.3`, `0.9.2`, `0.9.1`, `0.9.0`

## Default

The `<Textarea>` is the multiline version of the `<Input>`. It is a wrapper for the [HTML textarea](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/textarea).

<Canvas>
  <Story name="Default">
    <Textarea placeholder="e.g. Hello World" />
  </Story>
</Canvas>

You can set the height of the textarea using the `rows` prop.

<Canvas>
  <Story name="Height">
    <Textarea placeholder="3 lines of textarea" rows="3" />
  </Story>
</Canvas>

## Loading

You can control the loading state with the `loading` prop on the `<Control>` container. You can also add the `size` prop to the `<Control>` to change the loading icon size.

## Disabled

Add the `disabled` prop to create a disabled textarea.

## Readonly

If you add the `readOnly` prop, the textarea will look similar to a normal one, but is not editable and has no shadow.

<Canvas>
  <Story name="Readonly">
    <Textarea readOnly value="This content is readonly" />
  </Story>
</Canvas>
