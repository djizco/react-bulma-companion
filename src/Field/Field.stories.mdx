# Field

[Bulma Documentation](https://bulma.io/documentation/form/general/#form-field)

Supported Versions: `0.9.4`, `0.9.3`, `0.9.2`, `0.9.1`, `0.9.0`

## Default

The `<Field>` is a simple container for:

- `<Label>`: a text label
- `<Control>`: a form control
- `<Help>`: an optional help text

<Canvas>
  <Story name="Default">
    <Field>
      <Label>Label</Label>
      <Control>
        <Input placeholder="Text Input" />
      </Control>
      <Help>This is help text.</Help>
    </Field>
  </Story>
</Canvas>

This container allows form fields to be **spaced consistently**.

<Canvas>
  <Story name="Spaced Fields">
    <React.Fragment>
      <Field>
        <Label>Name</Label>
        <Control>
          <Input placeholder="e.g. Alex Smith" />
        </Control>
      </Field>
      <Field>
        <Label>Email</Label>
        <Control>
          <Input type="email" placeholder="e.g. alexsmith@gmail.com" />
        </Control>
      </Field>
    </React.Fragment>
  </Story>
</Canvas>

## Disabled Form

You can disable part or a whole form by wrapping a set of controls in a `<fieldset>` with the `disabled` prop.

<Canvas>
  <Story name="Disabled Form">
    <fieldset disabled>
      <Field>
        <Label>Name</Label>
        <Control>
          <Input placeholder="e.g. Alex Smith" />
        </Control>
      </Field>
      <Field>
        <Label>Email</Label>
        <Control>
          <Input type="email" placeholder="e.g. alexsmith@gmail.com" />
        </Control>
      </Field>
    </fieldset>
  </Story>
</Canvas>
