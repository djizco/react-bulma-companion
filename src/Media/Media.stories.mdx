# Media

[Bulma Documentation](https://bulma.io/documentation/layout/media-object/)

Supported Versions: `0.9.1`, `0.9.0`

## Default

The [media object](http://www.stubbornella.org/content/2010/06/25/the-media-object-saves-hundreds-of-lines-of-code) is a UI element perfect for repeatable and nestable content.

You can include *any* other Bulma element, like inputs, textareas, icons, buttons... or even a **nav bar**.
