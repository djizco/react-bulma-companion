# Table

[Bulma Documentation](https://bulma.io/documentation/elements/table/)

Supported Versions: `0.9.1`, `0.9.0`

## Default

You can create a **Bulma table** with the `Table` component and the following Subcomponents:

`Table.Head`, `Table.Foot`, `Table.Body`, `Table.Row`, `Table.HeadCell`, `Table.DataCell`, `Table.Container`
