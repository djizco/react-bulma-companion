# Dropdown

### An interactive **dropdown menu** for discoverable content

[Bulma Documentation](https://bulma.io/documentation/components/dropdown/)

Supported Versions: `0.9.4`, `0.9.3`, `0.9.2`, `0.9.1`, `0.9.0`

## Default

The `<Dropdown>` is a container for a dropdown button and a dropdown menu.

- `<Dropdown>`: the main container
  - `<Dropdown.Trigger>` the container for a button
  - `<Dropdown.Menu>` the toggable menu, hidden by default
    - `<Dropdown.Content>` the dropdown box, with a white background and a shadow
      - `<Dropdown.Item>` each single item of the dropdown, which can either be a a or a div
      - `<Dropdown.Divider>` a horizontal line to separate dropdown items

## Dropdown Content

While the `<Dropdown.Item>` can be rendered as a link with `component="a"`, the default `<div>` allows almost **any type of content**.

Toggle the `active` prop to make the dropdown content display.

<Canvas>
  <Story name="Content">
    <div className="has-text-centered" style={{ minHeight: 280 }}>
      <Dropdown active>
        <Dropdown.Trigger>
          <Button aria-haspopup="true" aria-controls="dropdown-menu-2">
            <span>Content</span>
            <Icon size="small">
              <FontAwesomeIcon icon={faAngleDown} />
            </Icon>
          </Button>
        </Dropdown.Trigger>
        <Dropdown.Menu id="dropdown-menu-2" role="menu">
          <Dropdown.Content>
            <Dropdown.Item>
              <p>You can insert <strong>any type of content</strong> within the dropdown menu.</p>
            </Dropdown.Item>
            <Dropdown.Divider />
            <Dropdown.Item>
              <p>You simply need to use a default dropdown item.</p>
            </Dropdown.Item>
            <Dropdown.Divider />
            <Dropdown.Item component="a">
              This is a link
            </Dropdown.Item>
          </Dropdown.Content>
        </Dropdown.Menu>
      </Dropdown>
    </div>
  </Story>
</Canvas>

## Hoverable

Add the `hoverable` prop to make the dropdown show up when **hovering** the `<Dropdown.Trigger>`

## Right aligned

You can add the `right` prop to have a **right-aligned** dropdown. The dropdown menu will line up with the right side of the dropdown trigger.

<Canvas>
  <Story name="Right Aligned">
    <div className="has-text-centered" style={{ minHeight: 150 }}>
      <Dropdown right active>
        <Dropdown.Trigger>
          <Button aria-haspopup="true" aria-controls="dropdown-menu-6">
            <span>Right aligned</span>
            <Icon size="small">
              <FontAwesomeIcon icon={faAngleDown} />
            </Icon>
          </Button>
        </Dropdown.Trigger>
        <Dropdown.Menu id="dropdown-menu-6">
          <Dropdown.Content>
            <Dropdown.Item>
              <p>Add the <code>right</code> prop for a <strong>right-aligned</strong> dropdown.</p>
            </Dropdown.Item>
          </Dropdown.Content>
        </Dropdown.Menu>
      </Dropdown>
    </div>
  </Story>
</Canvas>
