# Menu

### A simple **menu**, for any type of vertical navigation

[Bulma Documentation](https://bulma.io/documentation/components/menu/)

Supported Versions: `0.9.1`, `0.9.0`

## Default

The `<Menu>` is a vertical navigation component that comprises:

- the `<Menu>` container
- informative `<Menu.Label>` labels
- interactive `<Menu.List>` lists that can be nested up to 2 levels
- `<Menu.ListItem>` as children of menu lists
- `<Menu.Link>` as content for the list items
