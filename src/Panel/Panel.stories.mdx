# Panel

### A composable **panel**, for compact controls

[Bulma Documentation](https://bulma.io/documentation/components/panel/)

Supported Versions: `0.9.3`, `0.9.2`, `0.9.1`, `0.9.0`

## Default

The panel structure is as follows:

- `<Panel>`
  - `<Panel.Heading>`
  - `<Panel.Block>`
    - `<Panel.Icon>`
  - `<Panel.Tabs>`
    - `<Panel.Tab>`

These examples use [React Fontawesome](https://github.com/FortAwesome/react-fontawesome) with Font Awesome icons.

## Panel Block Types

The `<Panel.Block>` is rendered as a `<div>` by default.

To make it a link use `component="a"`. It can also be a **label** with a checkbox inside by using `component="label"`.

The `<Panel.Block>` can contain other elements such as `<Control>`, `<Input>` or `<Button>`.
