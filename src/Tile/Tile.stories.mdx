# Tile

### A **single tile** element to build 2-dimensional Metro-like, Pinterest-like, or whatever-you-like grids

[Bulma Documentation](https://bulma.io/documentation/layout/tiles/)

Supported Versions: `0.9.2`, `0.9.1`, `0.9.0`

## Example

To build intricate 2-dimensional layouts, you only need a **single element**: the `<Tile>`

## Modifiers

The tile comes with several modifiers props.

- `type`: `ancestor`, `parent`, or `child`.
- `size`: `1` - `12`
- `color` and `light`: uses the notification class under the hood to alter the tile colors.
- `vertical`: to create a vertical tile
- `box`: add the box styling to a tile
