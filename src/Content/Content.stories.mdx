# Content

### A single class to handle **WYSIWYG** generated content, where only **HTML tags** are available

[Bulma Documentation](https://bulma.io/documentation/elements/content/)

Supported Versions: `0.9.4`, `0.9.3`

## Default

When you can't use the CSS classes you want, or when you just want to directly use HTML tags, use the `<Content>` component as container. It can handle almost any HTML tag:

- `<p>` paragraphs
- `<ul>` `<ol>` `<dl>` lists
- `<h1>` to `<h6>` headings
- `<blockquote>` quotes
- `<em>` and `<strong>`
- `<table>` `<tr>` `<th>` `<td>` tables

This `<Content>` component can be used in any context where you just want to (or can only) write some **text**.

## Ordered Lists

**Ordered lists**, `ol` tags, support different **types** of items markers. To modify them, use either:

- the corresponding HTML attribute value
- one of the following CSS modifier classes: `is-lower-alpha`, `is-lower-roman`, `is-upper-alpha` or `is-upper-roman`

<Canvas>
