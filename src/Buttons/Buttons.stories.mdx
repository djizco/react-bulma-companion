# Buttons

[Bulma Documentation](https://bulma.io/documentation/elements/button/#list-of-buttons)

Supported Versions: `0.9.2`, `0.9.1`, `0.9.0`

## Default

The `<Buttons>` container creates a list for your buttons.

<Canvas>
  <Story name="Button List">
    <Buttons>
      <Button color="success">Save Changes</Button>
      <Button color="info">Save and continue</Button>
      <Button color="danger">Cancel</Button>
    </Buttons>
  </Story>
</Canvas>

## Size

You can change the size of **multiple buttons** at once by applying a `size` prop:

<Canvas>
  <Story name="Sizes">
    <Buttons size="medium">
      <Button>All</Button>
      <Button>Medium</Button>
      <Button>Size</Button>
    </Buttons>
  </Story>
</Canvas>

## Long List

If the list is very long, it will automatically wrap on multiple lines, while keeping all buttons evenly spaced.

<Canvas>
  <Story name="Long List">
    <Buttons>

      <Button>Two</Button>
      <Button>Three</Button>
      <Button>Four</Button>
      <Button>Five</Button>
      <Button>Six</Button>
      <Button>Seven</Button>

      <Button>Twelve</Button>
      <Button>Thirteen</Button>

      <Button>Sixteen</Button>
      <Button>Seventeen</Button>
      <Button>Eighteen</Button>
      <Button>Nineteen</Button>
      <Button>Twenty</Button>
    </Buttons>
  </Story>
</Canvas>

## Addons

You can attach buttons together with the `addons` prop.

<Canvas>
  <Story name="Addons">
    <Buttons addons>
      <Button>Yes</Button>
      <Button>Maybe</Button>
      <Button>No</Button>
    </Buttons>
  </Story>
</Canvas>

## Alignment

Use the `align` prop to alter the alignment.

<Canvas>
  <Story name="Alignment">
    <React.Fragment>
      <Buttons addons>
        <Button>Yes</Button>
        <Button>Maybe</Button>
        <Button>No</Button>
      </Buttons>
      <Buttons align="center" addons>
        <Button>Yes</Button>
        <Button>Maybe</Button>
        <Button>No</Button>
      </Buttons>
      <Buttons align="right" addons>
        <Button>Yes</Button>
        <Button>Maybe</Button>
        <Button>No</Button>
      </Buttons>
    </React.Fragment>
  </Story>
</Canvas>
