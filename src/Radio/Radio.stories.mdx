# Radio

### The mutually exclusive **radio buttons** in their native format

[Bulma Documentation](https://bulma.io/documentation/form/radio/)

Supported Versions: `0.9.4`, `0.9.3`, `0.9.2`, `0.9.1`, `0.9.0`

## Default

The `<Radio>` is a simple **container** around the `<input type="radio">` [HTML element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/radio). It is intentionally not styled, to preserve cross-browser compatibility and the user experience.

Make sure the linked radio buttons have the same value for their `name` HTML attribute.

<Canvas>
  <Story name="Default">
    <Control>
      <Radio>
        <input type="radio" name="default" />
        <span>{' Yes'}</span>
      </Radio>
      <Radio>
        <input type="radio" name="default" />
        <span>{' No'}</span>
      </Radio>
    </Control>
  </Story>
</Canvas>

## Checked

You can check a radio button by default by adding the `checked` HTML attribute to the `<input>` element.

<Canvas>
  <Story name="Checked">
    <Control>
      <Radio>
        <input type="radio" name="answer" />
        <span>{' Yes'}</span>
      </Radio>
      <Radio>
        <input type="radio" name="answer" onChange={() => {}} checked />
        <span>{' No'}</span>
      </Radio>
    </Control>
  </Story>
</Canvas>

## Disabled

You can disable a radio button by adding the `disabled` HTML attribute to both the `<Radio>` and the `<input type="radio">`.

  <Canvas>
  <Story name="Disabled">
    <Control>
      <Radio>
        <input type="radio" name="rsvp" />
        <span>{' Going'}</span>
      </Radio>
      <Radio>
        <input type="radio" name="rsvp" />
        <span>{' Not going'}</span>
      </Radio>
      <Radio disabled>
        <input type="radio" name="rsvp" disabled />
        <span>{' Maybe'}</span>
      </Radio>
    </Control>
  </Story>
</Canvas>
