# Icon

### Bulma is compatible with **all icon font libraries**

[Bulma Documentation](https://bulma.io/documentation/elements/icon/)

Supported Versions: `0.9.1`, `0.9.0`

## Default

The `<Icon>` component is a **container** for any type of **icon font**. Because the icons can take a few seconds to load, and because you want control over the **space** the icons will take, you can use the `Icon` component as a reliable square container that will prevent the page to "jump" on page load.

These examples use [React Fontawesome](https://github.com/FortAwesome/react-fontawesome) with Font Awesome icons.

<Canvas>
  <Story name="Default">
    <Icon>
      <FontAwesomeIcon icon={faHome} />
    </Icon>
  </Story>
</Canvas>

By default, the `<Icon>` container will take up exactly `1.5rem x 1.5rem`. The icon itself is sized accordingly to the icon library you're using. For example, Font Awesome 5 icons will **inherit** the font size.

## Sizes

The `<Icon>` container comes in 4 sizes by use of the `size` prop. Reference the [Bulma Documentation](https://bulma.io/documentation/elements/icon/#sizes) to see how they work with different icon sizes. The background

The background is only here for demonstration purposes, to highlight the icon container's area.
