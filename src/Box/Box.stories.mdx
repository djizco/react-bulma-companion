# Box

[Bulma Documentation](https://bulma.io/documentation/elements/box/)

Supported Versions: `0.9.1`, `0.9.0`

## Default

The `<Box>` component is simply a container with a shadow, a border, a radius, and some padding.
For example, you can include a media object:

<Canvas>
  <Story name="Default">
    <Box>Temporary Content</Box>
  </Story>
</Canvas>

## Link

It can be rendered as an `<a>` link by adding the `link` prop.

<Canvas>
  <Story name="Link">
    <Box link>Temporary Content</Box>
  </Story>
</Canvas>
