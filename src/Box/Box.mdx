# Box

### A white **box** to contain other elements

[Bulma Documentation](https://bulma.io/documentation/elements/box/)

## Default

The `<Box>` component is simply a container with a shadow, a border, a radius, and some padding.
For example, you can include a media object:

<Canvas>
  <Story name="Default">
    <Box>
      <Media>
        <Media.Left>
          <Image size="64x64">
            <Image.Content src="https://bulma.io/images/placeholders/128x128.png" />
          </Image>
        </Media.Left>
        <Media.Content>
          <Content>
            <p>
              <strong>John Smith</strong> <small>@johnsmith</small> <small>31m</small>
              <br />
              Lorem ipsum dolor sit amet, consectetur adipiscing elit. Aenean efficitur sit amet massa fringilla egestas. Nullam condimentum luctus turpis.
            </p>
          </Content>
          <Level mobile>
            <Level.Left>
              <Level.Item component="a">
                <Icon size="small">
                  <FontAwesomeIcon icon={faReply} />
                </Icon>
              </Level.Item>
              <Level.Item component="a">
                <Icon size="small">
                  <FontAwesomeIcon icon={faRetweet} />
                </Icon>
              </Level.Item>
              <Level.Item component="a">
                <Icon size="small">
                  <FontAwesomeIcon icon={faHeart} />
                </Icon>
              </Level.Item>
            </Level.Left>
          </Level>
        </Media.Content>
      </Media>
    </Box>
  </Story>
</Canvas>
