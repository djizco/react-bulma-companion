# Button

### The classic **button**, in different colors, sizes, and states

[Bulma Documentation](https://bulma.io/documentation/elements/button/)

Supported Versions: `0.9.1`, `0.9.0`

## Default

The `<Button>` component is an essential element of any design. It's meant to look and behave as an **interactive** element of your page.

<Canvas>
  <Story name="Default">
    <Button>Button</Button>
  </Story>
</Canvas>

## Types

The `<Button>` can be rendered as an `<a>` anchor with the `link` prop. You can make a submit button or reset button with the `type` prop.

<Canvas>
  <Story name="Types">

      <Button>Regular Button</Button>
      <Button type="submit">Submit Button</Button>
      <Button type="reset">Reset Button</Button>
    </Buttons>
  </Story>
</Canvas>

## Sizes

The `<Button>` comes in 4 different sizes by use of the `size` prop: small, normal, medium, large.

While the default size is the normal one, the `size="normal"` prop exists in case you need to reset the button to its normal size.

<Canvas>
  <Story name="Sizes">
    <Buttons>
      <Button size="small">Small</Button>
      <Button>Default</Button>
      <Button size="normal">Normal</Button>
      <Button size="medium">Medium</Button>
      <Button size="large">Large</Button>
    </Buttons>
  </Story>
</Canvas>

You can change the size of **multiple buttons** at once by wrapping them in a `Buttons` component, and applying a `size` prop:

<Canvas>
  <Story name="Group Sizes">
    <Buttons size="medium">
      <Button>All</Button>
      <Button>Medium</Button>
      <Button>Size</Button>
    </Buttons>
  </Story>
</Canvas>

You can change the size of only a **subset of buttons** by simply applying a `size` prop to them.
For example, if you want all buttons to be `small` but still have one in its `normal` size, simply do the following:

<Canvas>
  <Story name="Group Sizes with Subset">
    <Buttons size="small">
      <Button>Small</Button>
      <Button>Small</Button>
      <Button>Small</Button>
      <Button size="normal">Normal</Button>
      <Button>Small</Button>
    </Buttons>
  </Story>
</Canvas>

## Icons

Buttons can easily be enhanced by adding a **Font Awesome icon**. For the best results, wrap the inner text in a separate `span` element.

These examles use [React Fontawesome](https://github.com/FortAwesome/react-fontawesome) with Font Awesome icons.

<Canvas>
  <Story name="Icons">
    <Buttons>
      <Button>
        <Icon size="small">

        </Icon>
      </Button>

        <Icon size="small">
          <FontAwesomeIcon icon={faUnderline} />
        </Icon>
      </Button>

## Button Group

If you want to **group** buttons together on a **single line**, use the `grouped` prop on the `<Field>` container:

<Canvas>
  <Story name="Grouped">
    <Field grouped>
      <Control>
        <Button color="link">Save Changes</Button>
      </Control>
      <Control>
        <Button>Cancel</Button>
      </Control>
      <Control>
        <Button color="danger">Delete Post</Button>
      </Control>
    </Field>
  </Story>
</Canvas>
