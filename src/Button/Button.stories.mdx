# Button

[Bulma Documentation](https://bulma.io/documentation/elements/button/)

Supported Versions: `0.9.1`, `0.9.0`

## Default

The **button** is an essential element of any design. It's meant to look and behave as an **interactive** element of your page.

## Sizes

The button comes in 4 different sizes: small, normal, medium, large.

While the default size is the normal one, the `size="normal"` prop exists in case you need to reset the button to its normal size.

<Canvas>
  <Story name="Sizes">
    <Buttons>
      <Button size="small">Small</Button>
      <Button>Default</Button>
      <Button size="normal">Normal</Button>
      <Button size="medium">Medium</Button>
      <Button size="large">Large</Button>
    </Buttons>
  </Story>
</Canvas>

You can change the size of **multiple buttons** at once by wrapping them in a `Buttons` component, and applying a `size` prop:

<Canvas>
  <Story name="Group Sizes">
    <Buttons size="medium">
      <Button>All</Button>
      <Button>Medium</Button>
      <Button>Size</Button>
    </Buttons>
  </Story>
</Canvas>

You can change the size of only a **subset of buttons** by simply applying a `size` prop to them.
For example, if you want all buttons to be `small` but still have one in its `normal` size, simply do the following:

<Canvas>
  <Story name="Group Sizes with Subset">
    <Buttons size="small">
      <Button>Small</Button>
      <Button>Small</Button>
      <Button>Small</Button>
      <Button size="normal">Normal</Button>
      <Button>Small</Button>
    </Buttons>
  </Story>
</Canvas>

## Displays

You can add the `fullWidth` prop to a button to make it take up the full width of it's container.
