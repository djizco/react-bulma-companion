# Pagination

### A responsive, usable, and flexible **pagination**

[Bulma Documentation](https://bulma.io/documentation/components/pagination/)

Supported Versions: `0.9.1`, `0.9.0`

## Default

The Pagination structure is as follows:

- `<Pagination>`
  - `<Pagination.Previous>`
  - `<Pagination.Next>`
  - `<Pagination.List>`
    - `<Pagination.ListItem>`
      - `<Pagination.Link>`
      - `<Pagination.Ellipsis>`

All elements are optional so you can compose your pagination as you wish.

## States

You can disable some links if they are not active, or change the amount of page numbers available.

You can disable the `<Pagination.Previous>`,`<Pagination.Next>`, or `<Pagination.Link>` with the `disabled` prop.

You can also mark a `<Pagination.Link>` as the current page with the `current` prop.

<Canvas>
  <Story name="States">
    <Pagination>
      <Pagination.Previous disabled>Previous</Pagination.Previous>
      <Pagination.Next>Next</Pagination.Next>
      <Pagination.List>
        <Pagination.ListItem>
          <Pagination.Link current>1</Pagination.Link>
        </Pagination.ListItem>
        <Pagination.ListItem>
          <Pagination.Link>2</Pagination.Link>
        </Pagination.ListItem>
        <Pagination.ListItem>
          <Pagination.Link>3</Pagination.Link>
        </Pagination.ListItem>
      </Pagination.List>
    </Pagination>
  </Story>
</Canvas>
