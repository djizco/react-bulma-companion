# Container

### A simple **container** to center your content horizontally

[Bulma Documentation](https://bulma.io/documentation/layout/container/)

Supported Versions: `0.9.4`, `0.9.3`, `0.9.2`, `0.9.1`

## Default

The `<Container>` is a simple utility element that allows you to **center** content on larger viewports. It can be used in any context, but mostly as a **direct child** of either:

- `<Navbar>`
- `<Hero>`
- `<Section>`
- `<Footer>`

## Widescreen or FullHD Only

With the two props `breakpoint="widescreen"` and `breakpoint="fullhd"`, you can have a fullwidth container until those specific breakpoints.

## Desktop and Widescreen maximum widths

Sometimes, you might want a **narrow** container on larger viewports. That's why Bulma provides 2 modifiers:

- `breakpoint="max-desktop"` will behave like a desktop container
- `breakpoint="max-widescreen"` will behave like a widescreen container
