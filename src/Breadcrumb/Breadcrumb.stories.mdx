# Breadcrumb

[Bulma Documentation](https://bulma.io/documentation/components/breadcrumb/)

Supported Versions: `0.9.1`, `0.9.0`

## Default

The **Bulma breadcrumb** is a simple navigation component that only requires a `<Breadcrumb>` container and a `<Breadcrumb.List>`. The dividers are automatically created in the content of the ::before pseudo-element of the `<Breadcrumb.ListItem>`.

You can inform the current page using the `active` prop in a `<Breadcrumb.ListItem>`. It will disable the navigation of inner links.

  <Story name="Default">
    <Breadcrumb>
      <Breadcrumb.List>
        <Breadcrumb.ListItem><a>Bulma</a></Breadcrumb.ListItem>
        <Breadcrumb.ListItem><a>Documentation</a></Breadcrumb.ListItem>

        <Breadcrumb.ListItem active><a>Breadcrumb</a></Breadcrumb.ListItem>
      </Breadcrumb.List>
    </Breadcrumb>
  </Story>
</Canvas>
