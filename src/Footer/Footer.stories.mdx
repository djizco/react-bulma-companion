# Footer

### A simple responsive **footer** which can include anything: lists, headings, columns, icons, buttons, etc.

[Bulma Documentation](https://bulma.io/documentation/layout/footer/)

Supported Versions: `0.9.2`, `0.9.1`, `0.9.0`

## Default

The `<Footer>` is a simple container, with lots of bottom padding, making it great as the last element of any webpage.
